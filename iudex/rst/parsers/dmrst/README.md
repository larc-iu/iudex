# dmrst

DMRST joint segmentation + parsing model. Optionally trains a per-subtoken
EDU-boundary segmenter alongside the parser; otherwise runs in gold-EDU mode.

- **Paper:** Zhengyuan Liu, Ke Shi, Nancy F. Chen. *DMRST: A Joint Framework
  for Document-Level Multilingual RST Discourse Segmentation and Parsing.*
  CODI @ EMNLP 2021.
  [ACL Anthology](https://aclanthology.org/2021.codi-main.15)
- **Reference implementation:** <https://github.com/seq-to-mind/DMRST_Parser>
